# Sistema de Leilões

Este projeto é um sistema informatizado para gerenciar leilões.

## Funcionalidades

- Definir `cadastroVIEW` como a tela principal.
- Funcionalidade de salvar itens no banco de dados.
- Exibir mensagem de sucesso ou erro ao salvar itens.
- Listagem de itens cadastrados.

## Tecnologias Utilizadas

- Java
- MySQL
